# Adobe Integration for Soulvan CLI

This directory contains the integration for Adobe workflows within the Soulvan CLI project. It provides tools and configurations necessary for exporting Unity projects to Adobe applications.

## Overview

The Adobe integration allows users to seamlessly export their Unity projects, ensuring compatibility with Adobe's suite of creative tools. This integration is designed to streamline workflows for artists and developers working with cinematic scenes and high-fidelity rendering.

## Features

- **Export Presets**: Predefined settings for exporting projects to Adobe applications.
- **Integration with Unity**: Tools to facilitate the export process directly from the Unity Editor.
- **Support for 8K Rendering**: Capable of handling high-resolution exports for cinematic quality.

## Getting Started

1. **Install Dependencies**: Ensure that you have the necessary Adobe applications installed.
2. **Configure Export Settings**: Modify the `adobe-exports.json` file to customize your export presets.
3. **Use the CLI**: Utilize the `export-adobe` command in the Soulvan CLI to initiate the export process.
